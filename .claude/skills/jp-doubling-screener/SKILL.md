# JP Doubling Screener — 日本株「半年で2倍」候補の日次抽出

## 目的
直近5年で **126営業日(≒半年)以内に+100%** となった日本株（doublers）を分析し、
点火前に共通して現れる **本質的要素** をデータで検証して採用。現在の全上場銘柄を同じ要素で
機械採点し、**何個の本質的要素が当てはまるか** で順位付けする。Python が定量計算を担い、
Claude（このスキル）が定性統合して最終ウォッチリストを作る。**ランニングコスト0円。**

## 前提・制約
- データ: yfinance（無料）。ユニバース: JPX 公開リスト（`data/universe.csv`）。
- ネットワーク: routine 環境を **Custom**（`*.finance.yahoo.com` 等を許可）か **Full** にする
  （`docs/operations.md` 参照）。既定 Trusted では金融ホストが遮断される。
- ファンダ要素は **現時点スナップショット**（`.info`）のみ。過去点別データは無料で入手不可なため
  過去検証は限定的（テクニカル・マクロは価格履歴で検証済み）。
- 投資助言ではなく要素該当度のスクリーニング支援。

## 本質的要素カタログ（採点対象）
> しきい値・採否は `experiments/results/report.md`（識別力 AUC / 命中率 lift）で検証し、
> `src/stockfactor/score.py` の `RULES` を唯一の正とする。下表は人間向けの要約。
> **採点ルールを変更したら必ず experiment ワークフローで再検証すること。**

### テクニカル（点火前の価格・出来高構造）
- 52週高値圏（高値の90%以上）でのブレイク準備
- 出来高急増（直近5日が60日平均の1.5倍以上）
- 移動平均パーフェクトオーダー（25>75>200）・200日線上
- 初動モメンタム（3カ月で+10%以上、ただし未だ2倍前）
- ボラティリティ収縮（VCP的: 直近20日のボラ/60日のボラ ≤ 0.85）
- TOPIX 相対強さ（6Mで市場をアウトパフォーム）

### ファンダ×ミクロ（企業の質・成長・伸びしろ）
- 小型時価総額（500億円未満）= 2倍の伸びしろ
- 増収率（YoY 15%以上）／増益・黒字転換（20%以上）
- ROE 10%以上・マージン改善
- 割安成長（PSR 5倍以下 等、過度な割高でない）

### マクロ（市場・小型株レジーム）
- 市場（TOPIX）が200日線上＝上昇トレンド
- 小型株（グロース市場）が市場を6Mで上回る＝小型株物色

## 日次手順（このスキルが実行する）

1. **データ更新の確認**: `data/universe.csv` が無ければ
   `python -c "from src.stockfactor import universe as u; u.refresh()"`（月1で十分）。
2. **決定論採点を実行**:
   ```bash
   python run_daily.py            # 全銘柄（routine 本番）
   # 動作確認時は: python run_daily.py --limit 300
   ```
   → `output/daily_scores.csv`（全候補・要素内訳）と `output/daily_report.md`（上位30）を生成。
3. **定性統合（LLMパート＝あなたの仕事）**: `output/daily_scores.csv` の上位を読み、各銘柄について:
   - 充足要素の**内訳（テクニカル/ファンダ/マクロのバランス）**を見る。3群すべてに該当する銘柄を重視。
   - 不自然な値（急騰直後で既に2倍済み＝出遅れでない、出来高が異常な仕手的動き等）を除外。
   - セクター・テーマの妥当性、`hits` に挙がった要素が**互いに整合**しているかを確認。
   - 同種の過去 doubler（`experiments/results/doublers.csv`）との類似性があれば言及。
4. **ウォッチリスト出力**: `output/watchlist_<YYYY-MM-DD>.md` に上位10〜20銘柄を、
   - コード・銘柄名・終値・充足要素数（群別内訳）・該当要素・**一言の根拠**・**留意点**
   で表にまとめる。最後に「市場レジーム（マクロ）の所見」を1段落。
5. （任意）結果をコミット。

## 出力フォーマット例（watchlist）
```
# ウォッチリスト 2026-06-08（市場レジーム: TOPIX 200日線上＝順張り有利）
| code | 銘柄 | 終値 | 要素計 | T/F/M | 主な該当要素 | 根拠 | 留意点 |
|------|------|------|--------|-------|--------------|------|--------|
| 1234 | ... | 1500 | 9/14 | 5/3/1 | 52週高値ブレイク,出来高急増,増収,小型 | 高値圏で出来高を伴い初動、増収率高い | 既に+40%で出遅れ感やや薄 |
```

## 重要な運用ルール
- **しきい値の根拠はデータ**: 体感で閾値を動かさない。変更は score.py → experiment 再検証 → 反映。
- **2群以上に該当しない銘柄は原則採用しない**（テクニカルだけの急騰は誤検出が多い）。
- 既に半年で2倍**済み**の銘柄は「これから2倍」候補ではない。初動段階（過熱前）を重視。
