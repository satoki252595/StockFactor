# 検証結果: 日本株「半年で2倍」の本質的要素

> 実データ（yfinance, 直近約5年）で仮説を検証した結論。再現は GitHub Actions の `experiment`
> ワークフロー（無料）。生データは `experiments/results/`。

## 実験設定
- ユニバース: JPX 上場一覧から **500銘柄サンプル**（seed=42）。全 3,734 銘柄の縮小版。
- 2倍株定義: ある起点 t0 から **126営業日以内に終値+100%**（先読みはラベルのみ、要素は t0 までの過去のみ）。
- 正例: doubler の点火点 t0 の要素（167件）／負例: 将来126日で+50%にも届かなかった日の要素（534件）。
- 指標: **AUC**（0.5=無情報, 0.5から離れるほど識別力）と **lift**（正例命中率−負例命中率）。

## H2 ✅ 2倍株は十分存在する
- doubler イベント **167件**、2倍を経験した銘柄 **105 / 500（21%）**。統計に十分。
- 例（126日内の最大上昇率）: データセクション(3905) +345%、FRONTEO(2158) +232%、
  大阪チタニウム(5726) +170%、精工技研(6834) +156%、フォーサイド(2330) +129% など。

## H3 ⚠️ 当初仮説（米国流モメンタム/ブレイク型）は **棄却**された
点火点を「52週高値ブレイク・トレンド整列・ボラ収縮」で説明しようとしたが、**逆**だった。

| 当初仮説 | データの結論 | 根拠 |
| --- | --- | --- |
| 52週高値圏でブレイク | ❌ 逆。**高値から下落した位置**から発火 | dist_52w_high AUC=0.33（中央値 pos0.70 / neg0.87） |
| ボラ収縮(VCP) | ❌ 逆。**ボラ拡大** | vcp AUC=0.61（pos1.06 / neg0.92） |
| 移動平均トレンド整列 | ❌ 無情報 | sma_aligned AUC≈0.49 |
| 市場上昇トレンドが追い風 | ❌ むしろ弱い逆相関 | mkt_above_200 AUC=0.44 |
| 相対強さ(対TOPIX) | ❌ ほぼ無情報 | rs_6m AUC=0.51 |

## ✅ 実データで確定した「本質的要素」（採用＝score.py の RULES）
日本の半年2倍株は **「高ボラ × 高値から下落（出遅れ/押し目）× 出来高急増 × ボラ拡大 × 直近は投げ」**
という **反転（リバーサル）型** プロファイル。トレンドフォロー型ではない。

| 要素 | 定義 | AUC | lift | 解釈 |
| --- | --- | --- | --- | --- |
| **high_volatility** | ATR% ≥ 0.04 | **0.81** | **+0.45** | 動ける銘柄。最強の識別子 |
| **off_highs** | 終値 ≤ 52週高値×0.80 | 0.33 | +0.27 | 高値から下落・出遅れ位置 |
| **wide_range**(参考) | 直近60日の値幅/終値 | 0.79 | — | 高ボラの別証拠（atr_pctと相関大のため非採点） |
| **below_sma25** | 終値 ≤ 25日線 | 0.33 | +0.22 | 押し目/底値圏 |
| **volume_surge** | 5日出来高 ≥ 60日平均×1.3 | 0.60 | +0.22 | 点火の出来高 |
| **vol_expansion** | vcp ≥ 1.0 | 0.61 | +0.19 | ボラ拡大 |
| **recent_dip** | 直近1カ月リターン ≤ 0 | 0.37 | +0.17 | 反転直前の投げ/capitulation |

## R9 ✅ スコア（充足要素数）は正例で有意に高い
| 統計 | 正例(doubler点火) | 負例(非上昇) |
| --- | --- | --- |
| 平均 | **3.83** | 2.30 |
| 中央値 | **4** | 2 |
| p25 | 3 | 1 |

→ 充足要素数で順位付けする方式は妥当（**再設計前は 3.53 vs 3.94 で反予測的**だった＝当初仮説が誤り。
　検証に基づき要素を入れ替えた結果、正例が明確に上回るようになった）。

## マクロについて
本サンプルでは市場レジーム・小型株物色などマクロ要素の識別力は弱かった（AUC≈0.50）。
半年2倍は **個別銘柄の事象**（イベント・需給）が支配的と解釈。マクロは低ウェイトの参考に留め、
スキルの定性評価で「順張り/逆張りの地合い」程度に用いる。

## ファンダ×ミクロの扱い（重要な限界）
- **無料では決算の発表時点（point-in-time）データが入手できない**ため、ファンダ要素は過去検証できない。
- よってファンダ（小型・増収・増益・ROE・割安）は **フォワード適用のフィルタ**として
  `validated=False` で採点に加える（文献的根拠ベース）。過去AUCは未測定であることを明示。

## 重要な注意（誤用防止）
- 本要素は「2倍に**なった**銘柄の点火前の姿」を、非上昇銘柄と対比して抽出したもの。
  同じ姿でも**下落する銘柄も多い**（高ボラ・押し目はリスクも高い）。本ツールは
  「2倍**になり得る**セットアップ」のスクリーニングであり、勝率や下落リスクは別問題。
- 検証は500銘柄サンプル・単一期間。全銘柄・期間分割での再検証は今後の課題（R13）。
